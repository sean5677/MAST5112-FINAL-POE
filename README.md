 Chef's Menu App (TypeScript)

A React TypeScript application for managing and displaying restaurant menu items with filtering capabilities.
Features

 Completed Requirements

1. Home Screen with Average Prices
   - Displays average price for each course (Starter, Main, Dessert)
   - Shows complete menu overview
   - Displays item count per course

2. Separate Management Screen
   - Add new menu items with form validation
   - Edit existing menu items
   - Remove menu items from the list
   - All data stored in React state array

3. Guest View with Filtering
   - Filter menu items by course (All, Starter, Main, Dessert)
   - Clean display without management actions
   - Real-time item count display

4. TypeScript Implementation
   - Strongly typed components and props
   - Interface definitions for all data structures
   - Type safety throughout the application
